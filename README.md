# Chaos Launcher

Chaos Launcher is a desktop Minecraft launcher for Windows.

## Microsoft Account Login

Chaos Launcher supports Microsoft sign-in for legitimate Minecraft account authentication.

- OAuth Client ID: `38536923-8a70-4375-9a74-f91f5fda22c6`
- Auth flow: Microsoft OAuth + Xbox/Minecraft token exchange
- Purpose: sign in users and launch Minecraft instances

## Compliance Statement

Chaos Launcher:

- does not bypass Minecraft license checks
- does not bypass account security checks
- does not disable safety protections
- does not impersonate Mojang, Minecraft, Microsoft, or Xbox
