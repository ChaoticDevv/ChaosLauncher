# Privacy Policy

Last updated: 2026-02-18

## 1. Scope

This policy describes how Chaos Launcher handles data when you use the desktop application.

## 2. Data We Process

Chaos Launcher may process:

- Microsoft OAuth tokens required for login
- Minecraft profile data (UUID, username, skin/cape URLs)
- launcher settings and local instance metadata
- technical logs needed for diagnostics

## 3. Purpose of Processing

Data is used only to:

- authenticate the user
- launch and manage Minecraft instances
- provide account-related UI features
- troubleshoot launcher issues

## 4. Data Storage

- Data is stored locally on the user's device.
- Chaos Launcher does not sell personal data.
- Chaos Launcher does not use user data for advertising.

## 5. Third-Party Services

Chaos Launcher communicates with:

- Microsoft identity endpoints
- Xbox/Minecraft authentication endpoints
- Mojang/Minecraft metadata endpoints (version and assets)

Each third-party service is governed by its own policy and terms.

## 6. Security

Reasonable technical measures are applied to reduce unauthorized access risk. No method is 100 percent secure.

For privacy requests, contact: `chaoticdeveloperr@gmail.com`.
