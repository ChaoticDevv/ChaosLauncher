# Security Policy

## Supported Versions

Security updates are provided for the latest public build of Chaos Launcher.

## Reporting a Vulnerability

If you discover a security issue, report it privately:

- Email: `chaoticdeveloperr@gmail.com`
- Subject: `Chaos Launcher Security Report`

Please include:

- affected version
- reproduction steps
- proof of concept (if available)
- impact assessment

## Response Targets

- Initial response: within 72 hours
- Triage decision: within 7 days
- Fix timeline: depends on severity and complexity

## Responsible Disclosure

Do not publicly disclose details until a fix is released or coordinated disclosure is agreed.
